# 🎮 FocusGo — Telegram Bot

**Temptation Simulator в Telegram** — игра-симулятор борьбы с искушениями при изучении Go.

---

## 📖 Описание

FocusGo — это Telegram-версия игры **Temptation Simulator**, которая помогает мотивировать к изучению языка программирования Go через геймификацию.

Играйте прямо в Telegram — не нужно устанавливать Go, терминал или другие программы!

---

## 🚀 Быстрый старт

### 1. Получите токен бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Придумайте имя и username для бота
4. Скопируйте полученный токен

### 2. Настройте бота

```bash
# Экспортируйте токен
export TELEGRAM_BOT_TOKEN="YOUR_BOT_TOKEN_HERE"
```

Или создайте файл `.env`:
```bash
TELEGRAM_BOT_TOKEN=your_token_here
```

### 3. Запустите бота

```bash
cd focusgo
go run .
```

Или используйте скомпилированный бинарник:
```bash
./focusgo
```

### 4. Откройте бота в Telegram

Найдите вашего бота по username и нажмите `/start`

## 🎮 Как играть

### Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветственное сообщение |
| `/play` | Начать новую игру |
| `/profile` | Показать профиль игрока |
| `/skills` | Дерево навыков |
| `/quests` | Ежедневные квесты |
| `/stats` | Расширенная статистика |
| `/save` | Сохранить прогресс |
| `/load` | Загрузить сохранение |
| `/help` | Справка |

### Игровой процесс

1. **Начните игру** — `/play`
2. **Изучайте Go** — выбирайте "Учить Go" на 30 или 60 минут
3. **Сопротивляйтесь искушениям** — CapCut, видеоигры, соцсети и другие
4. **Выполняйте квесты** — 5 ежедневных заданий
5. **Улучшайте навыки** — 12 навыков в 4 категориях
6. **Сражайтесь с боссами** — финальная битва каждый день
